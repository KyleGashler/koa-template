# KOA Template

I wanted a simple starting point to go from whenever building a rest API with Koa so here it is!

## Tech used

-   KOA
-   TypeScript
-   Pino Logger

## How to get started

1. `npm install`
2. `docker-compose up -d`
3. `npm run build`
4. `npm run start`
